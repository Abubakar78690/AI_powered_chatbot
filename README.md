# 🤖 AI Powered Chatbot

An AI-powered chatbot web application built using React.js, Express.js, Firebase, and Groq API.
The chatbot allows users to interact with an intelligent assistant in real-time with fast AI-generated responses.
---
# Features
* Real-time AI chatbot interaction
* Efficient Error handling
* Fast AI responses using Groq API
* Firebase Authentication
* Firebase Firestore Database Integration
* Modern responsive UI
* REST API based backend architecture
* Secure environment variable management
* Full-stack deployment ready
---
# Tech Stack
## Frontend
* React.js
* Vite
* CSS / Bootstrap
* Axios
## Backend
* Node.js
* Express.js
* Groq API
* REST APIs
## Database & Authentication
* Firebase Firestore
* Firebase Authentication

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Abubakar78690/AI_powered_chatbot.git
cd AI_powered_chatbot
```
---
## 2️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd ../backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
GROQ_API_KEY=your_groq_api_key
FIREBASE_API_KEY=your_firebase_api_key
```

---

# ▶️ Running the Project

## Start Backend

```bash
cd backend
npm start
```

---

## Start Frontend

```bash
cd frontend
npm run dev
```
